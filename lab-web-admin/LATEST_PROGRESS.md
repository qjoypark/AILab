# Web管理后台最新开发进度

## 本次更新内容（2026-03-17）

### 🎉 新增功能

#### 1. 药品选择组件 ✨
**文件**: `src/components/MaterialSelector.vue`

**功能特性**:
- 弹窗式药品选择器
- 支持关键词、药品类型筛选
- 表格行点击选择
- 分页功能
- 可在多个页面复用（入库、出库、申请、盘点等）

**使用场景**:
- 入库单添加药品
- 出库单添加药品
- 申请单添加药品
- 盘点单添加药品

---

#### 2. 审批处理页面 ✨
**文件**: `src/views/approval/ApprovalTodo.vue`
**路由**: `/approval/todo`

**功能特性**:
- 待审批列表展示
- 申请详情查看（申请人、部门、用途、明细）
- 审批处理：
  - 选择通过/拒绝
  - 填写审批意见
  - 修改批准数量（可小于申请数量）
- 审批提交

**业务流程**:
1. 查看待审批申请列表
2. 点击"审批"查看详情
3. 修改批准数量（可选）
4. 选择审批结果（通过/拒绝）
5. 填写审批意见
6. 提交审批

---

#### 3. 出库管理页面 ✨
**文件**: `src/views/inventory/StockOutManagement.vue`
**路由**: `/inventory/stock-out`

**功能特性**:
- 出库单列表（单号、仓库、类型、金额、状态）
- 出库类型：
  - 领用出库
  - 报废出库
  - 盘亏出库
- 创建出库单：
  - 选择仓库
  - 选择出库类型
  - 添加出库明细（使用药品选择组件）
  - 填写数量、批次号
- 出库单确认（扣减库存）
- 出库单详情查看
- 出库单删除（仅待确认状态）

**业务流程**:
1. 创建出库单
2. 选择仓库和出库类型
3. 添加药品明细
4. 保存出库单
5. 确认出库（更新库存）

---

#### 4. 库存盘点页面 ✨
**文件**: `src/views/inventory/StockCheckManagement.vue`
**路由**: `/inventory/stock-check`

**功能特性**:
- 盘点单列表（单号、仓库、类型、状态）
- 盘点类型：
  - 全盘
  - 抽盘
- 创建盘点单
- 录入盘点明细：
  - 添加药品（使用药品选择组件）
  - 填写批次号
  - 录入账面数量
  - 录入实际数量
  - 自动计算差异
  - 填写差异原因
- 盘点差异标记：
  - 盘盈（绿色）
  - 盘亏（红色）
- 完成盘点（调整库存）
- 盘点单详情查看

**业务流程**:
1. 创建盘点单（选择仓库、盘点类型）
2. 录入盘点明细
3. 添加药品并填写账面数量和实际数量
4. 系统自动计算差异
5. 保存盘点明细
6. 完成盘点（根据差异调整库存）
